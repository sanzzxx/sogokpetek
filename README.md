## UPGRADE FOR DEBIAN
```
apt update -y && apt upgrade -y && apt dist-upgrade -y && reboot
```

## UPGRADE FOR UBUNTU
```
apt update && apt upgrade -y && update-grub && sleep 2 && reboot
```

## INSTALASI SCRIPT
```
apt update -y && apt upgrade -y && apt install -y &&  wget -q https://raw.githubusercontent.com/sanzzxx/sogokpetek/main/setup.sh && chmod +x setup.sh && ./setup.sh
```
